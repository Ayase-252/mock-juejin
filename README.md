# A copy of Juejin

A copy of Juejin with [Vue.js](https://github.com/vuejs/vue)



## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
